The slides and exercises presented here are based on Ensembl 92
Regulatory Features:
  1. Print a list of epigenomes that are in the current regulatory build.
  2. Fetch the regulatory feature with stable id ENSR00000060807.
    Print
    • The name of its feature type,
    • The name of the sequence it is on and
    • The start and end coordinates.
      Hint: The sequence is stored in the Slice object.
  3. For every epigenome in the current regulatory build, print the activity that it is predicted to have in this epigenome.
Motif Features:
  1. Retrieve all Motif Features that have been annotated in the Regulatory Feature with stable id ‘ENSR00000000667’

  2. If the motif sequence changes due to a mutation you want to know how this would affect the binding affinity. Computing this is beyond the scope of the course, but the way the score of the binding matrix changes could give a pointer.

    In the above Regulatory Feature you have retrieved one with id ‘759795’. Compute how the score from the binding matrix changes, if the motif sequence changes to ’CCCGGCCCCACCCC’.

    Hint:
    • The motif object has a binding_matrix method. This can be used to compute the score from the binding matrix via the "relative_affinity" method.
Microarrays:

  1. Export all microarray platforms that are annotated for humans in Ensembl and their associated information.

  2. You have performed a microarray experiment with the array HumanWG_6_V2.

    The following probes gave you a positive signal:

    • ILMN_1763508,
    • ILMN_1861090,
    • ILMN_1890175,
    • ILMN_1749304,
    • ILMN_1894173,
    • ILMN_1911643,
    • ILMN_1891089,
    • ILMN_1859810,
    • ILMN_1843473 and
    • ILMN_1770856

    Which transcripts do they map to?

  3. Which genes do these transcripts belong to?
    Modify the script above so it retrieves the gene stable id for every transcript.
  4. For ILMN_1763508 from the same array, find all matches it makes in the human genome.
eQTLs:
  1. List all tissues currently available for human.
  2. Examine the variant rs2736340 using the variation endpoint.
  3. Find all genes where there is a significant hit (p-value < 0.005) and list their description (xref).
